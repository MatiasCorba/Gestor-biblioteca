# <u>Sistema de Gestión de Biblioteca</u>

## <u>Descripción</u>
Aplicación desarrollada en Python para administrar el inventario de una biblioteca, registrar usuarios y gestionar préstamos de libros.

__Estado:__ En desarrollo

## <u>Historia y Objetivos</u>
Este proyecto forma parte de mis primeros trabajos de desarrollo y tiene como objetivo acompañar mi formación como desarrollador Full Stack. Planeo ampliarlo y mejorarlo a medida que adquiera nuevos conocimientos durante el máster
__<u>Objetivos:</u>__
- Practica de python
- Aplicar programación modular
- Utilización de git y github
- Contruir un proyecto evolutivo para un portfolio

## <u>Tecnologias</u>
__V1.0:__ 
- Python
- Git
- GitHub

## <u>Funcionalidades</u>
__V1.0:__
- Registrar libros
- Registrar usuarios
- Gestionar prestamos
- Devolver libros

## <u>Estructura del proyecto</u>
- Biblioteca/
  - main.py
  - libros.py
  - usuarios.py
  - prestamos.py
